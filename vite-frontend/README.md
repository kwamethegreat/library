# IT Course Library — Vite + React + TypeScript

A modern responsive website for an online IT course library. It includes:

- Vite + React + TypeScript setup
- Searchable course library
- Category filters
- Featured course cards
- Learning path section
- Fully responsive CSS
- No required UI framework or external styling library

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Main files

```text
src/App.tsx              Main React page and components
src/data/courses.ts      Course data
src/types.ts             TypeScript course types
src/styles.css           Full responsive styling
```

## Customize

Edit `src/data/courses.ts` to add your real courses, categories, durations, ratings, and learning outcomes.
